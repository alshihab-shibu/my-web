# Alshihab Shibu - Portfolio Website

A responsive GitHub Pages portfolio showcasing Magento development expertise and software engineering work.

## 🌐 Live Demo
Visit: [https://alshihab-shibu.github.io/my-web/](https://alshihab-shibu.github.io/my-web/)

## 🚀 Features
- Responsive portfolio with modern layout
- SEO-friendly meta tags and structured data
- Accessible markup and skip link support
- Contact form with basic validation
- Lightweight static HTML, CSS, and JavaScript

## 🛠️ Project Files
- `index.html` — main homepage content
- `style.css` — responsive styles and animations
- `script.js` — interactive behaviors and form handling
- `images/` — profile and project visuals
- `README.md` — project documentation

## ▶️ Run Locally
```bash
cd d:\my-web
python -m http.server 8000
```
Then open `http://localhost:8000` in your browser.
